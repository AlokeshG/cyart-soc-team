# Evidence Preservation

This folder contains tasks related to evidence collection and preservation during security incidents.

Contents:
- Collection of volatile and non-volatile data (e.g., memory dump, logs)
- Documentation of evidence details (source, date, collected by)
- Hashing of evidence (e.g., SHA256) to maintain integrity
- Chain of custody records
- Tools used such as Velociraptor and FTK Imager

This section demonstrates the process of collecting and preserving digital evidence in a secure and reliable manner for investigation purposes.
