# Alert Triage

This folder contains tasks related to alert triage in a Security Operations Center (SOC).

Contents:
- Alert analysis and prioritization
- Identification of false positives and true positives
- Threat intelligence validation using tools like VirusTotal and AlienVault OTX
- Classification of alerts based on severity and impact
- Documentation of triage results

This section demonstrates the process of analyzing and validating security alerts to determine appropriate response actions.
