# Capstone Project: Full Alert-to-Response Cycle

This folder contains the complete simulation of a security incident from detection to response.

Project Overview:
This project demonstrates the full lifecycle of a security incident in a SOC environment, including attack simulation, detection, triage, response, and reporting.

Contents:
- Attack Simulation using a vulnerable system (e.g., Metasploitable2)
- Detection of attack using monitoring tools (e.g., Wazuh)
- Alert triage and classification based on severity
- Incident response actions (system isolation, IP blocking)
- Evidence collection and documentation
- Final incident report and stakeholder briefing

Objectives:
- Understand real-world SOC workflow
- Practice identifying and responding to cyber attacks
- Improve documentation and reporting skills

Outcome:
This project showcases the ability to handle a complete security incident efficiently, from initial alert generation to final resolution and reporting.
